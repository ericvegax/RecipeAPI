# RecipeAPI
A light-weight API to create &amp; add recipes 

## Usage

## Maven
```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```
```xml
<dependency>
    <groupId>com.github.realericvega</groupId>
    <artifactId>recipeAPI</artifactId>
    <version>1.0.0</version>
</dependency>
```
